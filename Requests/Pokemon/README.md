# Pokédex CLI (Python + PokeAPI)

A simple command-line Pokédex built with Python that fetches Pokémon data from the [PokeAPI](https://pokeapi.co/).  
It displays basic information such as name, height, Pokédex ID, and types, with colored output for better readability.

---

## ✨ Features
- Fetch Pokémon data by name using the PokeAPI.
- Displays:
  - Name
  - Height
  - Pokédex ID
  - Types
- Colored terminal output using ANSI escape codes.
- Error handling for invalid Pokémon names.
